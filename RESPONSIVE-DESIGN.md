# 響應式設計說明

## 概述
紐西蘭葡萄酒學院平台已針對手機、平板和桌面電腦進行全面響應式優化，確保在各種設備上都能提供最佳的學習體驗。

## 支援的設備類型

### 📱 手機 (< 768px)
- **最小寬度**: 320px
- **測試設備**: iPhone SE, iPhone 12/13/14, Samsung Galaxy S20/S21
- **方向**: 支援直向和橫向

### 📱 平板 (768px - 1024px)
- **測試設備**: iPad, iPad Pro, Samsung Galaxy Tab
- **方向**: 支援直向和橫向

### 💻 桌面電腦 (> 1024px)
- **推薦解析度**: 1920×1080, 2560×1440
- **瀏覽器**: Chrome, Firefox, Safari, Edge

## 響應式斷點

```css
/* 平板 */
@media (max-width: 1024px) { ... }

/* 手機/小平板 */
@media (max-width: 768px) { ... }

/* 小手機 */
@media (max-width: 480px) { ... }
```

## 主要功能優化

### 1. 觸控友好設計
- ✅ 所有按鈕最小高度 44px (符合 Apple HIG 標準)
- ✅ 測驗選項在手機上為單欄佈局，按鈕高度 48-52px
- ✅ 導航控制按鈕在小螢幕上橫跨全寬
- ✅ 移除長按選擇高亮效果 (`-webkit-tap-highlight-color: transparent`)

### 2. 地圖整合
#### 嵌入式地圖 (課程投影片中)
- **桌面**: 左右並排 (文字 50% | 地圖 50%)
- **平板**: 文字/地圖垂直排列，地圖高度 60vh
- **手機**: 地圖優先顯示 (order: -1)，高度 450px (768px) / 350px (480px)

#### 全螢幕地圖 (酒區探索)
- **桌面**: 左側區域列表 (320px) | 右側地圖
- **手機**: 上方區域列表 (40%) | 下方地圖 (60%)

### 3. 導航系統
#### 課程導航 (CourseNavigation)
- **桌面**: 水平排列 (Logo | Level 選擇 | 進度)
- **平板/手機**: 
  - Logo 和返回按鈕在上方
  - Level 選擇器換行至下方，支援橫向滾動
  - 進度徽章保持在右上角

#### 投影片導航 (SlideViewer)
- **桌面**: 固定底部工具列
- **手機**: 
  - 工具列換行
  - 投影片控制 (上一頁/下一頁/進度) 獨立一行並橫跨全寬
  - 按鈕間距增大便於點擊

### 4. 測驗系統
#### QuizViewer (課內測驗)
- **桌面**: 2 欄選項佈局
- **手機**: 單欄佈局，每個選項 min-height: 52px

#### ReviewQuiz (複習測驗)
- **桌面**: 自適應多欄佈局
- **手機**: 
  - 單欄佈局
  - 下一題按鈕橫跨全寬
  - 完成後的操作按鈕垂直堆疊

### 5. 投影片內容
#### 文字內容
- **桌面**: font-size: 20px, line-height: 1.8
- **平板**: font-size: 18px, line-height: 1.7
- **手機**: font-size: 17px (768px) / 16px (480px)

#### 圖表和表格
- **時間軸、循環圖**: 自動縮小間距和字體
- **比較表格**: 在手機上可橫向滾動
- **信息圖**: 圖片和文字垂直堆疊

### 6. 輸入欄位優化
- **字體大小**: 最小 16px (防止 iOS Safari 自動縮放)
- **最小高度**: 44px (手機) / 40px (小手機)
- **觸控區域**: 增加內邊距確保易於點擊

## 性能優化

### 滾動優化
```css
-webkit-overflow-scrolling: touch;
```
啟用硬體加速滾動，提供更流暢的體驗

### 觸控反饋
```css
touch-action: manipulation;
```
減少點擊延遲，提升觸控響應速度

### 地圖優化
- 在手機上減少地圖初始高度
- 使用 `max-height` 限制地圖不超過可視區域
- 自動調整地圖邊距以適應小螢幕

4. **測驗功能**
   - 選項選擇

### 測試工具

- **真實設備**: 推薦在實際手機和平板上測試
- **線上工具**: BrowserStack, LambdaTest

## 已知限制

1. **地圖圖層**
   - 在極小螢幕 (< 360px) 上，地圖控制按鈕可能略顯擁擠
   - 建議使用 >= 375px 寬度的設備以獲得最佳體驗

2. **橫向模式**
   - 手機橫向模式下，可視區域高度受限
   - 建議在直向模式下進行課程學習

3. **舊版瀏覽器**
   - 不支援 IE11 及以下版本
   - 需要現代瀏覽器支援 CSS Grid, Flexbox, CSS Variables

### 視口設定
```html

```

- `.content-map-layout`: 地圖/文字並排佈局
- `.quiz-options`: 測驗選項網格
- `.slide-nav-controls`: 投影片導航控制
- `.utility-bar`: 工具列

### 受影響的組件
1. `App.vue` - 主應用佈局
2. `SlideViewer.vue` - 投影片顯示
3. `QuizViewer.vue` - 課內測驗
4. `ReviewQuiz.vue` - 複習測驗
5. `CourseNavigation.vue` - 課程導航
6. `NewZealandMap.vue` - 地圖組件
7. `AppLayout.vue` - 側邊欄佈局
