# 紐西蘭葡萄酒教學專案 🍷

一個基於 WSET(Wine & Spirit Education Trust)架構的互動式紐西蘭葡萄酒教育平台,結合互動地圖與系統化課程內容。

## 專案特色

### 🎓 系統化課程架構
- **6大核心模組**,涵蓋紐西蘭葡萄酒的各個面向
- 遵循 WSET 國際認證標準
- 從基礎概論到進階品酒技巧

### 🗺️ 互動式產區地圖
- 整合 Mapbox GL JS 的動態地圖
- 17個主要產區的 GeoJSON 數據
- 點擊產區查看詳細資訊

### 📚 豐富的學習內容
- 文字課程 + 互動地圖課程
- 產區風土特色介紹
- 葡萄品種深入解析
- 品酒技巧與餐酒搭配

### ✅ 學習進度追蹤
- 自動記錄學習進度
- 視覺化進度條
- 課程完成標記

## 課程大綱

### 模組 1: 紐西蘭葡萄酒概論
- 紐西蘭葡萄酒簡史
- 氣候與地理特徵
- 產業法規與分級制度

### 模組 2: 主要葡萄品種
- Sauvignon Blanc - 紐西蘭的驕傲
- Pinot Noir - 優雅的紅色珍寶
- Chardonnay 與其他白葡萄
- 其他紅葡萄品種

### 模組 3: 北島產區探索
- Hawke's Bay - 紅酒重鎮
- Wairarapa/Martinborough
- Auckland 及周邊產區
- Gisborne 與其他北島產區

### 模組 4: 南島產區探索
- Marlborough - 白酒之都
- Central Otago - 世界最南端產區
- Nelson 與 Canterbury
- Waitaki Valley

### 模組 5: 葡萄栽培與釀造
- 葡萄園管理實踐
- 永續與有機栽培
- 釀造技術與風格
- 橡木桶使用與陳年

### 模組 6: 品酒技巧與餐酒搭配
- WSET 系統化品酒法
- 紐西蘭葡萄酒風格分析
- 餐酒搭配原則
- 品酒筆記與評估

## 循序漸進 (Levels)

- 本專案支援以 Level (階級) 方式學習：Level 1（基礎）、Level 2（中階）、Level 3（進階）。
- 左側導航可選擇 Level，並只顯示該 Level 的模組與課程，方便循序學習與分級教材設計。
- 你可以從 Level 1 開始，完成後再逐步解鎖 Level 2/3 的內容。

## 新增功能

- **循序解鎖**: Level 2/3 會在使用者完成前一級所有課程後自動解鎖，未解鎖時相關按鈕與模組會顯示鎖定提示。
- **Map-First 模式**: 新增地圖模式切換按鈕，可將介面切換為以地圖為主並將課程側欄縮為浮動面板，適合以地理探索為導向的學習。
- **互動地圖焦點**: 當課程為「互動地圖」類型時，系統會自動切換到地圖模式並將地圖聚焦到該課程指定的產區。

## 技術架構

### 前端技術
- **Vue 3** - 漸進式 JavaScript 框架
- **Composition API** - 現代化的組件邏輯組織
- **Mapbox GL JS** - 互動式地圖渲染
- **Vite** - 快速的開發構建工具

### 數據結構
- GeoJSON 格式的產區地理數據
- JSON 格式的課程結構與產區資訊
- HTML 格式的課程內容
